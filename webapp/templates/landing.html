Karunya University — Micro Project

Thermal Profiling of
Electronic Components

Simulate heat diffusion across CPU and GPU chips using partial differential equations. Visualize thermal hotspots and evaluate cooling strategies in real time.

Launch App View on GitHub
🔥

PDE Heat Simulation

Solves the 2D heat diffusion equation using finite-difference method with TDP-based thermal conductivity.

💾

18 CPU/GPU Profiles

Pre-loaded profiles for Intel, AMD, Apple Silicon, Nvidia, Qualcomm and server-grade processors.

📡

Live Sensor Feed

Connect your phone as a live sensor — accelerometer data is mapped to temperature and fed into the simulation in real time.

📊

3D Thermal Visualization

Interactive 3D surface heatmap and cross-section gradient chart powered by Plotly.

🤖

AI Cooling Recommendations

Intelligent suggestions based on simulated temperature — from thermal paste to liquid cooling.

🗄️

Simulation History

Every simulation is saved to a PostgreSQL database hosted on Render for analysis and comparison.
